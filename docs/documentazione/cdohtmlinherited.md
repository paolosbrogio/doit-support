# cDoHtmlInherited

> Migrato da: https://sites.google.com/site/doitsup/documentazione/cdohtmlinherited
> Data migrazione: 2026-05-06

Imposta i tag html degli oggetti DoiT.

## Oggetti applicabili

La feature `cDoHtmlInherited` puo' essere utilizzata sui seguenti oggetti:

- `cDoPage.cDoHtmlInherited`
- `cDoSplitter.cDoHtmlInherited`
- `cDoSection.cDoHtmlInherited`
- `cDoElement.cDoHtmlInherited`
- `cDoHtmlInheritedColumnHeader` (solo per elementi delle sezioni di tipo grid)
- `cDoHtmlInheritedTableCell` (solo per elementi delle sezioni di tipo grid)
